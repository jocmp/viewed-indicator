# Viewed Indicator


## Getting Started

<!-- Try out the demo on [CodeSandbox](https://codesandbox.io/p/github/jocmp/viewed-indicator/main) -->

Or run it locally:

1. Clone this repo
2. Install the dependencies
   ```
   npm install
   ```
3. Run the server
   ```
   npm run start
   ```
4. Open your browser and navigate to http://localhost:3000
